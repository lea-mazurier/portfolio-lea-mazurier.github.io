Food Truck

Ce projet est le premier site web que j'ai réalisé de l'identité visuelle jusqu'au développement et au maquettage. La première partie de ce projet food-truck fait parti de la SAE 105 durant laquelle nous devions produire un site web pour un food-truck inventé de toutes pièces par nous-même. Le site web est un site statique, seule la page du menu est dynamique. Il a été codé avec les langages html et css ainsi que javascript pour la page dynamique.

Etant mon premier site web jamais réalisé je n'avais l'expérience, il y a donc certains éléments qui ne me plaise pas et que je changerais si je devais le refaire. Comme par exemple la disposition des éléments dans la page Menu.

Wireframe

Pour ce site j'ai réalisé les wireframe de chaque page ainsi que l'arbre logique de la page d'accueil.

L'identité visuelle

Pour l'identité visuelle du food-truck j'ai réalisé un moodboard, un menu et un visuel du camion, ils ont été réalisés à l'aide de Adobe Illustrator. Pour ce food-truck je voulais rester sur une identité visuelle assez sombre avec seulement quelques touches de jaune et de rouge pour faire quelque chose de plutôt luxueux.

Le logo

Voici deux idées de logo pour le food-truck. Le logo retenu est le logo 2, pour autant le logo 1 n'a pas été totalement abandonné car la pâte apparait en mascotte au dessus du camion dans le visuel de celui-ci. Réalisés avec Illustrator.

La communication

Pour finir ce projet j'ai réalisé deux idées de post pour la communication du food-truck. Tous cela a été réalisé à l'aide du logiciel Adobe Illustrator.
