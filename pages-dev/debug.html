Bug Tracker

Le but de la SAE 203 était de créer un site de bugtracker. Pour cela il y a 3 rôles différents un administrateur, des testeurs et des développeurs. Ce site a été réalisé grâce au langage html pour les formulaires et php pour tout le reste.

Cette SAE m'a permis de mettre en pratique et regrouper tous ce que j'avais appris durant cette première année.

base de données + MCD et MLD

Voici les 2 tables de la base de données MySQL ainsi que le MCD et le MLD de celle-ci.

Page d'authentification

Pour l'authentification dans le site on récupère tout d'abord les login et les mots de passe stockés dans la base de données grâce à une requête sql et ceux rentrés dans le formulaire.

Puis on vérifie si les login et mot de passe rentrés dans le site correspondent à ceux de la base de données pour ensuite envoyer sur la bonne page et créer une session avec les if else.

La connexion à la base de données a été faite avec l'extension PDO qui est inclus dans PHP.

Page testeur + affichage tickets

Etant donné que l'affichage des tickets se fait sur chaque page j'ai utilisé la fonction include de PHP afin de ne pas avoir à écrire le même code sur chaque page. De plus cela me permet de modifier plusieurs pages en ne modifiant qu'un seul code.
